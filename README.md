### 📢 إدارة القناة
| الأمر | الوصف | الصلاحية |
|-------|-------|----------|
| `.نشر [نص]` | نشر منشور في القناة | مطور |

## ➕ إضافة أوامر جديدة

### الطريقة السهلة:
1. افتح ملف `plugins/example/template.js`
2. انسخه لأي مجلد داخل `plugins/`
3. عدل الاسم والأوامر والكود
4. **البوت هيشيله تلقائياً!** مش محتاج ريستارت

### مثال - أمر بسيط:
```js
const myCmd = async (m, { conn, bot, text }) => {
    await m.reply(`مرحباً يا ${m.pushName}! كتبت: ${text}`);
};

myCmd.command = ["سلام", "hi"];
myCmd.usage = ["سلام"];
myCmd.category = "example"; // channel | contest | admins | owner | info | example
myCmd.owner = false;
myCmd.cooldown = 2000;
myCmd.disabled = false;

export default myCmd;
```

### الفئات المتاحة:
| الفئة | الظهور في القائمة |
|-------|-----------------|
| `channel` | 📢 إدارة القناة |
| `contest` | 🏆 المسابقات |
| `admins` | 👥 الجروبات |
| `owner` | ⚔️ المطورين |
| `info` | 🗃️ معلومات |
| `example` | ✳️ مثال |
